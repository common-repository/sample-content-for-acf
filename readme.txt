== Description ==

Quickly populate Advanced Custom Fields with lorem ipsum text, placeholder stock images and more.

**This plugin requires at least ACF Pro 5.8.**
If you don't already own [ACF Pro](https://www.advancedcustomfields.com/pro/), you should consider it. It's one of the most powerful WordPress plugins available.

== How It Works ==

When using this plugin there will be an eyedropper icon next to each ACF label in the admin that when clicked, will instantly auto-populate that field with sample content. 

**This applies to all ACF Pro field types!**

* Auto-populate image fields with stock/placeholder imagery
* Auto-populate text fields with generated lorem ipsum
* Auto-populate wysiwyg fields with lorem ipsum paragraphs
* Auto-choose random posts & taxonomies on relational fields
* Auto-choose values on Advanced fields 
* Auto-choose choices from choice fields
* Auto-populate multiple rows of fields on Repeaters **(PRO)**
* Auto-populate all layouts on Flexible Content fields **(PRO)**
* Auto-populate all fields in Groups **(PRO)**
* Auto-populate Galleries with placeholder images **(PRO)**
* Auto-populate wysiwyg fields with rich content **(PRO)**

Field validation rules are taken in to a count and will be respected when using the auto-populator.

== Installation ==

1. Activate the plugin
2. Navigate to "Sample Content" settings which can be found on the left-hand menu of your admin 
3. Either use the search prompts to import sample imagry or choose your own in the Sample Image Inventory
4. Repeat step 3 for videos and files if you wish

And that's it! From here it is happy auto-populating!

== Usage ==

There will be an eyedropper icon next to each ACF label in the admin that when clicked, will instantly auto-populate that field with sample content. 

**Eyedropper Icon Locations**

* Most fields: Next to the ACF field label
* Field groups *(PRO)*: When hovering over the field group meta box, the eyedropper icon will appear in the top right of the title bar
* ACF Layout *(PRO)*: When hovering over the layout's title bar, the eyedropper icon will appear in the top right next to the other default ACF action icons

== Frequently Asked Questions ==
= How does it work with groups? =
When populating groups with sample content, it populates all subfields recursively within the group with one click!
= How does it work with repeaters? =
When populating repeaters with sample content, it will first add a few rows to you repeater field, then auto-populate all subfields recursively within each row - with one click!
= How does it work with flexible content? =
When populating a flexible content field, the plugin intelligently will find all of your configured layouts for your flexible content field, then adds each one AND auto-populates all of the subfields within each layout recursively.
